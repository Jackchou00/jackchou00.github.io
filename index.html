光电舜宇实习

光电学院大三短学期：光电专业实习 为什么选择专业实习 TODO 7.3 Day01 入住电子厂、准备打白工 学院预定的校车，九点半从西迁碑到余姚，路上两小时左右基本都在睡觉。先是开进城中村送兰江厂区的同学下车，路边停的车少有浙B牌照，反倒是豫、皖这两地的车牌居多，电线杆上贴满了工厂的招工广告，多是些装配、注塑的岗位，待遇一般是包吃住、月薪5000-7500，工作时长和环境就不得而知了，想想这个薪资待遇远超往年前来实习的学长，没想到傍晚更有惊喜。说回校车，校车在狭小的街道上兜了两圈，终于确定把兰江的几位同学放下，过了没一会儿又传来说兰江宿舍压根没有这几位同学的信息，随行的老师便又是一番折腾，车行过兰江园区，仿佛同电子厂无异。 又是一阵颠簸，行至舜宇基地大门，门卫却是一问三不知，再是一番沟通，原来得先去三里地外的宿舍办理入住，这宿舍乃是农贸市场街边商铺上盖，大巴载着最后两位同学扬长而去，留下狭小的宿管办外，十几号人和各自的行李，烈日下矗立着上交身份证等着办理入住，颇有几分进厂打工的架势。宿管阿姨好一阵手忙脚乱，终于是签字画押，成功入住大学生宿舍。宿舍两人间，房间不小，一人一床一桌，满是浮灰，似是许久未住人。不过好在终于有落脚之处，简单休息，已是一点出头，赶紧楼下兰州拉面炒饭果腹。 午睡醒来，惊觉已经五点出头，手机里，群聊热烈讨论着薪资问题，虽然不多，但也是这二十天的回报，忽闻今年经济形势下滑，这实习岗位都是学院办公室苦苦求来，更别说薪资问题，直接是一笔勾销，向学院确认，果真如此，顿时心灰意冷，好好的短学期，不仅下乡打工，还落得个无偿奉献，实属不易。又是兰州拉面一顿，毕竟没了工资，可不能再挥霍。 饭后正是日落时分，晚霞满天，甚是震撼。之后洗澡摸鱼，早早准备入睡，待明日早起报道，是办公室还是电子厂，明日见分晓。

解决换行符导致的hugo主题渲染问题

问题描述 由Hugo生成的静态网页在本地运行时正常，将Public目录push到Github后，Github Pages无法正确显示CSS样式。 解决 这一问题是由于Windows和MacOS或linux系统的换行符不同导致的，而Git会自动将换行符转换为CRLF，因此在Windows上生成的静态网页在Github上无法正确显示。 解决办法是将git的换行符转换功能关闭，即: git config --global core.autocrlf false 再次push到Github上，即可正常显示。
