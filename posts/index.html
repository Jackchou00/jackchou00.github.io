光电学院大三短学期：光电专业实习 为什么选择专业实习 TODO 7.3 Day01 入住电子厂、准备打白工 学院预定的校车，九点半从西迁碑到余姚，路上两小时左右基本都在睡觉。先是开进城中村送兰江厂区的同学下车，路边停的车少有浙B牌照，反倒是豫、皖这两地的车牌居多，电线杆上贴满了工厂的招工广告，多是些装配、注塑的岗位，待遇一般是包吃住、月薪5000-7500，工作时长和环境就不得而知了，想想这个薪资待遇远超往年前来实习的学长，没想到傍晚更有惊喜。说回校车，校车在狭小的街道上兜了两圈，终于确定把兰江的几位同学放下，过了没一会儿又传来说兰江宿舍压根没有这几位同学的信息，随行的老师便又是一番折腾，车行过兰江园区，仿佛同电子厂无异。 又是一阵颠簸，行至舜宇基地大门，门卫却是一问三不知，再是一番沟通，原来得先去三里地外的宿舍办理入住，这宿舍乃是农贸市场街边商铺上盖，大巴载着最后两位同学扬长而去，留下狭小的宿管办外，十几号人和各自的行李，烈日下矗立着上交身份证等着办理入住，颇有几分进厂打工的架势。宿管阿姨好一阵手忙脚乱，终于是签字画押，成功入住大学生宿舍。宿舍两人间，房间不小，一人一床一桌，满是浮灰，似是许久未住人。不过好在终于有落脚之处，简单休息，已是一点出头，赶紧楼下兰州拉面炒饭果腹。 午睡醒来，惊觉已经五点出头，手机里，群聊热烈讨论着薪资问题，虽然不多，但也是这二十天的回报，忽闻今年经济形势下滑，这实习岗位都是学院办公室苦苦求来，更别说薪资问题，直接是一笔勾销，向学院确认，果真如此，顿时心灰意冷，好好的短学期，不仅下乡打工，还落得个无偿奉献，实属不易。又是兰州拉面一顿，毕竟没了工资，可不能再挥霍。 饭后正是日落时分，晚霞满天，甚是震撼。之后洗澡摸鱼，早早准备入睡，待明日早起报道，是办公室还是电子厂，明日见分晓。 7.4 Day02 入职、一分钱一分货 宿舍离厂区有接近两公里路，早上七点半就已经酷热难耐，于是四个人一起打车去的厂区 又是一通嘈杂混乱办理入职，领张工牌，待人齐了领到一边的培训室，一人发一个“入职礼包”，其实就两张纸一本笔记本一支笔，就这四样东西甚至还有漏装的。培训其实就是签实习协议书之类，顺便介绍一下余姚人才政策，又是一通签字，之后便通知到工作的地方去。当然是没人带路的，自己去厂区里找吧。 到了办公楼，再是一通混乱场景，终于与前来接应的员工接上头，电梯上各自楼层。舜宇光学的办公室就是其集团展示时候的大平层办公区，宽敞明亮，感觉还是不错。行政让签字带走一本新员工入职培训手册，里面是一些最基本的几何光学知识，还有些工艺流程、工作上的事情，比较偏向实用和制造。之后带到工位上，说给配了个台式机，一会儿送到。课长是谁谁谁（没听清），人现在不在，就走了。没想到这是今天最后一个和我打交道的舜宇员工，接下来的一天都是我在工位上摸鱼划水，没人管没人问，十分符合白嫖来的大学生黑工应尽的职责。 摸鱼到十一点，直接开溜去吃饭，食堂很近，一个个档口有不同的套餐售卖，价格不贵，一个豉油鸡饭只11块，刚好把饭补用掉，据说一餐饭两次饭补，也有刷了第二次就让补钱的。吃完回去，已经把灯全关了，午休的午休，摸鱼的摸鱼。摸一上午鱼已经是累的不行（bushi），开睡，下午一点行政会开灯。 下午一次去打水回来，台式机已经送到，让自己把线接起来。这台式一个侧透全塔机箱，里面能看见吹雪主板上一个硕大的猫头鹰D15，一通折腾，找行政要来密码，开机一看，D15下镇压的是一颗12900KF……这舜宇竟如此财大气粗？显卡是1650，16G内存，一张500G的980做的系统盘，装的是企业版Windows10，有一些远程管理软件。登上OA一看，啥权限都没有，网络也只开放到内网，也不知道给我发电脑干嘛，只能把这大家伙晾在一边。 在OA里逛了一圈，能访问到一些2020年以前的文档，他们的产品立项报告、DR（设计评审）报告等等，基本都是一些给国产手机做的前后摄像头光学部分，产量和份额相当惊人，基本都是一供，数量都在百万量级，一颗摄像头光学部分的BOM成本大概两三块。算是今天为数不多有意思的发现，应该不涉密。 再翻翻入职培训手册，其实挺实用，产品开发各个环节要注意什么，黑话是什么意思都有记录，对光学镜头来说，他们似乎尤其在乎杂光的分析和处理，占了相当大的比重。一下就五点了，本来六点后打下班卡能多一次晚饭的饭补，但是第一天打不了下班卡，五点掐点开溜，回宿舍沙县果腹。 晚上床上躺会儿就睡过去了，洗个澡，明天见。不知道能不能见着课长说上两句话，找不到也没事，白嫖的还想咋样。

解决换行符导致的hugo主题渲染问题

问题描述 由Hugo生成的静态网页在本地运行时正常，将Public目录push到Github后，Github Pages无法正确显示CSS样式。 解决 这一问题是由于Windows和MacOS或linux系统的换行符不同导致的，而Git会自动将换行符转换为CRLF，因此在Windows上生成的静态网页在Github上无法正确显示。 解决办法是将git的换行符转换功能关闭，即: git config --global core.autocrlf false 再次push到Github上，即可正常显示。
