我#

我是 Jack Chou。

我目前是色彩科学领域的研究生，关于我和我们的工作，请参阅 我的个人页面 和 实验室网站。任何有关色彩科学，影像和显示方面的疑问和交流，都欢迎通过各种方式联系我，我的邮箱是 jackchou00@zju.edu.cn。

一些有关色彩科学的小工具和项目可以在 GitHub 上找到，比如解析和处理高动态范围图像格式，用于显示或相机特征化的模型等。

爱好有很多，主要是摄影、猫和咖啡，摆弄一些小玩意儿。

这里#

这个博客第一次创建在 2023 年 6 月 29 日，当时的目的是在秋招的时候能够有一个地方展示自己的内容，实际上，以那时那种一知半解的经验、阅历和心态，并不能写出像样的东西，那次秋招现在想来也不可能成功。不过也有像 Processing Raw Images in MATLAB 这样有一点内容的东西，但更多是被我用 Draft 雪藏的文章。

后来，一个非常幸运的机会结束了那次诡异的秋招，也不再有什么动力和目标去更新这个博客，而是偶尔用在一些奇怪的用途。比如，它被用来给巴黎住 Airbnb 遇到的法国夫妻分享 明信片故事。

这是上一次建设博客的故事，之前的部分内容还能够在 古迹 🌃 中找到。

由于是一个部署在 Github 上的静态网站，我在 GitHub 上的其他 pages 也会拥有相同的域名，这样，访客如果不输入后面的内容，直接访问这个域名，就会看到混乱的，很久没有更新的博客。

而为了更方便的分享那些项目的 pages 和利用 CloudFlare 的 Tunnel 功能，我购买了 jackchou00.icu 这个域名，荒废博客的问题变得更加明显。

2025 年 1 月 20 日，为了满足我自己的分享欲和虚荣心，也为了让这个域名不要指向荒废的博客，重新装修过的 JacksBlog 正式 push 了。这一次，他的目的是记录一些科研和兴趣中的东西，分享一些图片和生活。

网页是由 Hugo 生成的，使用的主题是稍作修改的 PaperMod，部署在 Github Pages 上并由 Cloudflare DNS 和代理。

未来#

jackchou00.icu 域名到期之后，我选择更换成 jackchou.top 作为主域名，这个域名购买了十年，直到 2035 年，那时候，我们会在哪里呢？

另外，还有一些别的域名，比如 zms.im 和 miaosenzhou.com，暂时是重定向到这里。
